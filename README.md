翻譯內容還不完整，歡迎 PR 修正內容或協助翻譯  
  
![screenshot](./screenshot.png)

## 使用方法
- 下載 `strings.json`
- 取代 `%LOCALAPPDATA%\gitkraken\app-x.x.x\resources\app.asar.unpacked\src\strings.json`，`x.x.x` 為版本代號
